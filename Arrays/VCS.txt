VCS - Version Control System
(commits)
SVN, Subversion, Mercury, ..... Git/Github
Git - система контроля версий
GitHub - облачное хранилище исходный кодов,
		 которое поддерживает VCS Git

Репозиторий - это хранилище исхоных кодов.
.git

branch	- master
mainstream 

commit	- это контрольная точка, в которой сохраняются
		 все новые и измененные файлы репозитория.

push	- отправить commit с локального репозитория на удаленный.
fetch	- загрузить базу данных с изменениями с 
		  удаленного репозитория в локальный не распаковывая ее.
pull	- загрузить и распаковать изменения с удаленного 
		  репозитория в локальный.
