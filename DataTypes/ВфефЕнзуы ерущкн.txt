Переменная (Variable) - это именованная область памяти, содежимое которой
может изменяться в процессе выполнения программы.

//Синтаксис объявления переменной:
			type name;
type - тип переменной, он определяет 

1. Логические типы: bool (Boolean) 1 Byte
	true	- правда	1
	false	- ложь		0
2. Символьные типы: char (Character - Символ) 
	ASCII	
3. Числовые типы
	unsigned - можгут хранить лишь положительные целые числа.
	  signed - могут хранить как положительные, 
			   так и отрицательные целые числа.
			   int a;	//это будет знаковая переменная.
	  unsigned int b;	//беззнаковая переменная.
