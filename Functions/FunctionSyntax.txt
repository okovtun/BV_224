type name(parameters)
{
	....;
	group-of-statements;
	....;
}
type - тип возвращаемого значения
void - когда функция ничего НЕ возвращает

return value1, value2, value3;
return;
, - comma operator;

name - имя функции, нужно для того, чтобы к ней можно было 
	   обращаться по этому имени.

parameters - список принимаемых параметров функции.
Принимаемые параметры - это исходные данные для работы функции.
Принимаемые параметры - это самые обычные переменные, объявленные 
						в заголовке функции в ()

group-of-statements - тело функции, последовательность выражений к конструкций,
					  реализующая алгоритм, возложенный на функцию.
